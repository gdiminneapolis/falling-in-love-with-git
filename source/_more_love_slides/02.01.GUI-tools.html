GUI Tools

Alternative to using the Command line
There are several described on the Git site:

GUI Clients
GitHub Desktop -- GitHub's own desktop client
SourceTree -- A universal agnostic git desktop client from Atlassian
Your editor may also have GUI integration with Git and remote repository sites.
For example, WebStorm and other JetBrains IDEs have impeccable git and remote git integration.
Another impeccable example is Emacs' magit package.
Magit has the best interface for interactively squashing commits.
